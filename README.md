## Задание

Разработать сервис Web API на ASP NET Core по скачиванию перечня ООН из внешнего источника. Перечень доступен в формате XML. Необходимо распарсить xml, сохранить данные в базу PostgreSql.

Учесть, что после повторной загрузки перечня (если данное состояние перечня уже было загружено), данные снова парсить и сохранять в базу не надо. Необходимо сообщение в лог, что перечень уже был скачан.

 Также если при скачивании, парсинге, сохранении в БД возникла ошибка, то вся загрузка считается некорректной (все данные должны быть исключены из БД), должно быть сообщение в лог об ошибке.

Необходимые места должны быть покрыты тестами.

Функционал скачивания перечня:
POST запрос по адресу:  api/collectData

Ссылка на перечень ООН: https://scsanctions.un.org/resources/xml/ru/consolidated.xml

Итоговое решение залить на любой хостинг GIT репозиториев и выслать ссылку. Удостоверится, что репозиторий публичен.
